# 🌑 VANTA

Experimental sci-fi interface built with React, dynamic animations and ambient sound.

![Vanta Preview](./preview.png)

---

## ✨ Features

- Dynamic signal system
- Animated HUD interface
- Reactive hover effects
- Ambient sci-fi sound
- Procedural particles
- Animated orbit system
- Dynamic system messages
- Experimental visual atmosphere

---

## ⚛️ Built With

- React
- Vite
- JavaScript
- CSS animations

---

## 🌌 Concept

Vanta is an atmospheric interface experiment inspired by:
- futuristic operating systems
- sci-fi HUDs
- ambient digital spaces
- interactive art interfaces

The goal was to create a living interface with movement, sound and reactive visual behavior.
